## Strategy observations (logged at Wesley's request)

1. **The build:sell asymmetry is the bottleneck.** ~60 commits in the 2 weeks
   after the 23-Jun reframe declared B2B the A-priority, while deal follow-ups
   waited. Code threads reach done in days; deal threads stall at the
   human-follow-up step. Rule of thumb: when deals are in a waiting state, the
   itch to build should convert into *opening the next deal thread*, not a
   feature.
2. **Every deal arrived warm; there is no outbound motion.** Tersia→WBC,
   FIL→mSure, Blair inbound, Ghost called back. The brand opens doors — build
   pipeline so no single counterparty's silence stalls the whole motion.
3. **Follow-ups count as acting.** The 07-07 status (Ghost follow-up, Tersia
   ask, Blair pending, mSure consciously parked) is real deal work, not
   "sitting on hands".
